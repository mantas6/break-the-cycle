# break-the-cycle

This template should help get you started developing with Vue 3 in Vite.

## Project Setup

### Local nodejs installation

#### Install dependencies

```sh
npm install
```

#### Compile

```sh
npm run dev
```

#### Run Vitest watcher

```sh
npm test
```

### Using Docker

#### Install dependencies

```sh
./build install
```

#### Compile

```sh
npm run dev
```

#### Test

```sh
./build npm test
```
